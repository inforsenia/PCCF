## UP01: INTRODUCCIÓN A LOS SISTEMAS DE GESTIÓN EMPRESARIAL
### 1. Identificación
| Campo | Detalle |
| :--- | :--- |
| **Código** | UP01 |
| **Módulo** | Sistemas de gestión empresarial (0491) |
| **Duración** | **10 Horas** |
| **Temporalización** | Del **08/09/2025** al **15/09/2025** |

### 2. Fundamentación
**Resultado de Aprendizaje**
*   **RA01.** Identifica sistemas de planificación de recursos empresariales y de gestión de relaciones con clientes (ERP-CRM) reconociendo sus características y verificando la configuración del sistema informático.

**Criterios de Evaluación**
*   **a)** Se han reconocido los diferentes sistemas ERP-CRM que existen en el mercado.
*   **b)** Se han identificado los diferentes tipos de licencia de los sistemas ERP-CRM.
*   **c)** Se han comparado sistemas ERP-CRM en función de sus características y requisitos.
*   **d)** Se ha identificado el sistema operativo adecuado a cada sistema ERP-CRM.
*   **e)** Se ha identificado el sistema gestor de datos adecuado a cada sistema ERP-CRM.
*   **f)** Se han verificado las configuraciones del sistema operativo y del gestor de datos para garantizar la funcionalidad del ERP-CRM.
*   **g)** Se han documentado las operaciones realizadas.
*   **h)** Se han documentado las incidencias producidas durante el proceso.

**Competencias**
*   **Profesionales:** o), c), p).
*   **Ocupación:** u), q), r).

### 3. Organización
**Contenidos**
*   Concepto de ERP y CRM: características, módulos y arquitectura.
*   Principales sistemas ERP-CRM del mercado: Odoo, SAP, Microsoft Dynamics.
*   Tipos de licencia: privativa, libre y SaaS.
*   Requisitos hardware y software: sistemas operativos y gestores de bases de datos compatibles.
*   Verificación de configuraciones del sistema para la instalación de un ERP-CRM.
*   Documentación de operaciones e incidencias.

**Metodología**
*   Uso de metodologías activas centradas en el estudiante mediante el **proyecto OdooEmpresa**. El alumnado actúa como equipo técnico externo contratado para implantar un sistema ERP-CRM en una empresa ficticia, comenzando por el análisis del mercado y la elección justificada del sistema.

**Secuencia (Fases)**
*   **E1:** Estudio comparativo de sistemas ERP-CRM del mercado: características y licencias.
*   **E2:** Análisis de requisitos: elección y justificación del sistema para la empresa ficticia.
*   **E3:** Verificación del entorno: sistema operativo, gestor de datos y configuración previa.
*   **E4:** Entrega: informe técnico de selección y verificación del entorno documentado.

**Recursos**
*   Aula de informática, VirtualBox, sistema operativo Linux/Windows, documentación oficial de Odoo, fichas comparativas de ERP-CRM.
